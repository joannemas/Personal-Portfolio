mes découvertes:

HTML

La création de page web

Je travaille à créer entièrement des sites avec le HTML pour maitriser les bonnes structures et mettre en forme correctement différents éléments, en y ajoutant des images, des liens, combinant différents langages et des ressources externes.

CSS

Styliser les pages web

Avec le CSS je peux styliser les pages que j'ai pu réaliser et modifier à ma guise n'importe quel élément. J'utilise les notions de flexbox, grid je peux compléter avec Sass pour améliorer ces notions.
Je touche également aux animations et au responsive.

SASS

Styliser encore mieux

Pour styliser mes pages plus rapidement et avec de l'efficacité, j'utilise le préprocesseur SASS, ce qui simplifie mon travail et ma syntaxe. Il est ainsi encore plus pratique de créer de jolies pages, ou de respecter au mieux certaines maquettes.

JAVASCRIPT

Animer les pages web

Pour rendre un site plus vivant, j'apprends à utiliser Javascript dans des situations de. Pour cela, j'utilise principalement des modifications aux clics qui pourront servir par exemple lors d'un menu burger ou encore de simples actions pour interagir avec un site.

Tailwind

Styliser avec efficacité

J'ai la capacité d'utiliser un framework CSS tel que Tailwind afin de styliser et personnaliser mon espace de travail pour créer des pages rapidement directement à partir d'un fichier HTML.

PHP

Dynamiser des sites

Rendre un site continuellement à jour et dynamique est une condition où je travaille avec PHP, notament pour des parties blogs. J'ai la capacité de crée un espace de communication et d'échange, de modifier l'expérience de navigation d'un site en fonction de l'utilisateur et interagir avec une base de données.

MySQL

Gérer des bases de données

Afin d'utiliser pleinement le PHP, je travaille sur la gestion de base de données pour stocker des données en SQL et pouvoir les réutiliser. Je peux donc gérer des membres, des messages et commentaires, des droits d'accès, des demandes de contact.

Python

Programmer et automatiser

Mes compétences en Python me permettent de créer des algorithmes pour automatiser certaines tâches. Avec mes connaissances qui regroupent l'utilisation de modules, je peux créer des jeux textuels, des générateurs, des convertisseurs et un robot pouvant répondre.

Git

Gérer des versions

Pour concevoir plusieurs versions d'un site j'ai la capacité d'utiliser git. J'ai la possibilité également de gérer le versioning avec une équipe sur un même projet.

WordPress

Utiliser un CMS

J'ai la possibilité de créer rapidement un site Wordpress, ainsi que de développer des thèmes ou travailler sur un thème existant afin de concevoir un site personnalisé.

Creative Cloud

Créer avec Adobe

Les outils de la suite Adobe permettent de réaliser de nombreaux visuels. Je suis capable de les utiliser pour de nombreuses réalisations, en particulier Photoshop, XD et After Effects.
